[Desc] luci-sample-app01

This is the description page for luci-sample-app01.
luci-sample-app01 combines the wi-fi, router and access point configuration interfaces introduced in my blog article into a single page.

[Implementation difficulty Lv.1]
This section allows you to modify the parameters in the OpenWrt network and wireless configuration files (/etc/config/network and /etc/config/wireless). In other words, it is a LuCI plugin that only manipulates the configuration files managed by the standard OpenWrt application.

What this app can do

  • Wi-Fi Settings
  • Router Settings
  • AP Settings
